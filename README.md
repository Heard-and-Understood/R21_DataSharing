# R21_DataSharing

Datashare collection for NIH R21CA259543: *Deep Neural Networks to Identify Moments of Human Connection in Cancer Communication*.

The code in this collection captures the work that was done for bimodal image + text classification
using the PCCRI connectional silences dataset. It should be noted that we began this work using
the term "Multi-Modal BERT", or "MMBERT", before discovering that an "MMBERT" modal already exists 
and is unrelated to our approach. A more accurate description of our approach is "bimodal BERT", 
but you'll see the old term used throughout these scripts.
